# Microsoft 365 Public Mail Protocol Samples

This repository contains sample code showing how to interact with Microsoft 365 using public mail protocols.

## [SMTP Console application (C#)](https://github.com/MWC-Developer/Public-Mail-Protocols/tree/main/SMTP/SMTPOAuthSample)

A console application written in C# showing how to authenticate using OAuth (supports application and delegate flows) and send an email.

## [SMTP PowerShell script](https://github.com/MWC-Developer/Public-Mail-Protocols/blob/main/SMTP/Send-MailWithSMTPOAuth.ps1)

A PowerShell implementation of the same code as used in the console application.
